# JabberPoint Startup

## Vereisten
- Java 17 of hoger (Java 11 werkt meestal ook voor deze basis)

## Project starten in Eclipse
1. Open Eclipse.
2. Kies **File > Import > Existing Projects into Workspace**.
3. Selecteer de map `jabberpoint`.
4. Klik op **Finish**.
5. Open `src/jabberpoint/App.java`.
6. Klik **Run As > Java Application**.

## Project starten met VSCode extensions
1. Download en installeer de Java Extension Pack in VSCode.
2. Open de map `jabberpoint` in VSCode.
3. Open `src/jabberpoint/App.java`.
4. Klik op de groene "Run" knop boven de `main` methode of gebruik de sneltoets `Ctrl+F5` (Windows/Linux) of `Cmd+F5` (Mac).
